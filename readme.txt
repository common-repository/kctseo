Easily search, export, and import options from the WordPress wp_options table using specific keywords

== Description ==

The WP Options Manager plugin allows users to easily search, export, and import options from the WordPress wp_options table based on specific keywords. Whether you're migrating settings between sites or troubleshooting options, this tool streamlines the management of WordPress options by enabling keyword-based searches and seamless export/import functionality. Ideal for developers and administrators looking for a quick way to manage site settings across installations.

Features

    Search Options by Keywords: Quickly search options in the WordPress wp_options table using specific keywords.
    Export Options: Export selected options based on search results to a JSON file for backup or migration.
    Import Options: Import options from a previously exported file to restore or transfer settings.
    User-Friendly Interface: Easy-to-use interface for managing complex WordPress settings.

Installation

    Download the plugin ZIP file.
    Upload the plugin to the /wp-content/plugins/ directory or use the Plugins > Add New > Upload Plugin feature in the WordPress dashboard.
    Activate the plugin through the Plugins menu in WordPress.
    Go to the WP Options Manager under Tools to start using the plugin.

Usage

    Navigate to Tools > WP Options Manager.
    Enter the keyword to search for specific options in the wp_options table.
    Select the options you want to export and click on the Export button to download a JSON file.
    To import options, click the Import button and upload the previously exported file.

Frequently Asked Questions
How do I search for specific options?

Simply navigate to the WP Options Manager page, enter a keyword, and the plugin will search for matching entries in the wp_options table.
Can I export only specific options?

Yes, after performing a keyword search, you can select the specific options you wish to export before generating the export file.
Is it safe to import options?

Yes, the import feature is designed to safely restore or migrate options from a valid JSON file. However, be sure to review the imported data to avoid overwriting critical settings.
